The server allows for testing of ESP8266 devices in client and server modes directly from your PC.

To run a server we only need only install Apache but to make it really useful we also need to install MySql for the database and php for the coding...

Luckily enough we can install all of these program in one go using one the available installers which also provide some basic administration... You need to choose one of these installers for the WiKi link below... I tend to change from time to time but mostly use WampServer and Easyphp at the moment... WiKi Page

Installing a Local Server...
  • Pick one from the WiKi and follow the instructions to install...
  • Windows Note! Avoid installing into "Program Files" or "Program Files (x86)", change it to C:\some_folder ...
  • Identify the document root folder name, we'll need this later...
  • When your happy everything is working move to next section...
Installing ESP8266 Web Site...
  • For reference, here's my GIT repo ESP Server
  • To download the code use this link: Master (save to your usual location)...
  • The download includes one folder named esp and a few sketch files...
  • Copy the esp folder to your document root folder *(see note below)
* Document root path examples for Windows: C:\wamp\www ... C:\easyphp\data\localweb ... C:\xampp\htdocs...

The ESP code requires an IP address to communicate with the web page so we need to set up one...
